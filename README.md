# Cyber Hunt 🚀

A space shooter game built with C# and Windows Forms.

## Features
- 3 progressive levels
- 4-direction player movement
- 3 enemy movement patterns
- Health system with health pills
- Sound effects
- OOP principles implementation

## How to Run
1. Clone the repository
2. Open CyberHunt.slnx in Visual Studio
3. Press F5 to run

## OOP Principles Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interface
